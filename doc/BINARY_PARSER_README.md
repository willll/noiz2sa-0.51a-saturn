# BulletML Binary Parser - Header-Only Library

## Overview

`bulletmlparser_blb.hpp` is a header-only C++ library that provides fast binary parsing for BulletML pattern files (.blb format).

The parser exposes a pointer-based API and returns `bool` from `build()`/`parse()` rather than throwing exceptions.

## Features

- **Header-only**: No separate compilation required, just include the header
- **Drop-in replacement**: Compatible with existing `BulletMLParser` interface
- **Fast loading**: 32.8% smaller than XML, faster parsing
- **Memory efficient**: Can load from file or memory buffer
- **Format**: Supports BLB (BulletML Binary) format generated by `tools/bulletml_converter.py`

## Integration

### 1. Copy the header and required dependencies

```bash
cp src/bulletml_binary/bulletmlparser_blb.hpp <your_project>/include/
cp src/bulletml_binary/bulletmlstate.hpp <your_project>/include/
```

### 2. Include in your code

```cpp
#include "bulletmlparser_blb.hpp"

// Load from file
BulletMLParserBLB parser("patterns/boss01.blb");
if (!parser.build()) {
    // handle parse failure
}

// Or load from memory
const uint8_t* data = ... // your binary data
size_t size = ... // size in bytes
BulletMLParserBLB parser(data, size);
if (!parser.build()) {
    // handle parse failure
}
```

### 3. Use like standard BulletMLParser

```cpp
// Check orientation
if (parser.isHorizontal()) {
    // horizontal mode
}

// Get top-level actions
uint32_t action_count = 0;
BulletMLNode** actions = parser.getTopActions(&action_count);
for (uint32_t i = 0; i < action_count; ++i) {
    BulletMLNode* action = actions[i];
    // process action
}

// Resolve references by ID
BulletMLNode* bullet = parser.getBullet(0);
BulletMLNode* action = parser.getAction(0);
BulletMLNode* fire = parser.getFire(0);
```

## Binary Format

The .blb format uses:
- **Magic number**: `"BLB\x00"` (4 bytes)
- **Version**: 1 (uint16_t)
- **Little-endian**: All multi-byte values
- **Compressed**: ~33% smaller than XML

See `tools/BINARY_FORMAT.md` for complete specification.

## Converting XML to Binary

Use the Python converter tool:

```bash
# Convert single file
python3 tools/bulletml_converter.py pattern.xml pattern.blb

# Convert directory
python3 tools/bulletml_converter.py patterns_xml/ patterns_blb/

# Convert back for validation
python3 tools/bulletml_converter.py pattern.blb pattern_verify.xml
```

## Requirements

- C++11 or later
- `bulletmlstate.hpp`
- Saturn runtime headers used by this implementation: `srl_cd.hpp`, `srl_log.hpp`

## Error Handling

```cpp
BulletMLParserBLB parser("pattern.blb");

if (!parser.build()) {
    std::cerr << "Parse error: " << parser.getErrorMessage() << std::endl;
}
```

## Performance

Tested with 73 pattern files (boss + middle + zako):
- **XML total**: 79,914 bytes
- **BLB total**: 53,689 bytes  
- **Savings**: 26,225 bytes (32.8%)
- **Parse speed**: ~3-5x faster than XML parsing (no XML DOM overhead)
