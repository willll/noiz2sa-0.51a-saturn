# BulletML Binary Parser - Header-Only Library

## Overview

`bulletmlparser_binary.hpp` is a header-only C++ library that provides fast binary parsing for BulletML pattern files (.blb format).

## Features

- **Header-only**: No separate compilation required, just include the header
- **Drop-in replacement**: Compatible with existing `BulletMLParser` interface
- **Fast loading**: 32.8% smaller than XML, faster parsing
- **Memory efficient**: Can load from file or memory buffer
- **Format**: Supports BLB (BulletML Binary) format generated by `tools/bulletml_converter.py`

## Integration

### 1. Copy the header to your project

```bash
cp src/bulletml_binary/bulletmlparser_binary.hpp <your_project>/include/
```

### 2. Include in your code

```cpp
#include "bulletmlparser_binary.hpp"

// Load from file
BulletMLParserBinary parser("patterns/boss01.blb");
parser.build();

// Or load from memory
const uint8_t* data = ... // your binary data
size_t size = ... // size in bytes
BulletMLParserBinary parser(data, size);
parser.build();
```

### 3. Use like standard BulletMLParser

```cpp
// Check orientation
if (parser.isHorizontal()) {
    // horizontal mode
}

// Get top-level actions
const std::vector<BulletMLNode*>& actions = parser.getTopActions();
for (BulletMLNode* action : actions) {
    // process action
}

// Get bulletMap, actionMap, fireMap for resolving references
// These are populated automatically during build()
```

## Binary Format

The .blb format uses:
- **Magic number**: `"BLB\x00"` (4 bytes)
- **Version**: 1 (uint16_t)
- **Little-endian**: All multi-byte values
- **Compressed**: ~33% smaller than XML

See `tools/BINARY_FORMAT.md` for complete specification.

## Converting XML to Binary

Use the Python converter tool:

```bash
# Convert single file
python3 tools/bulletml_converter.py pattern.xml pattern.blb

# Convert directory
python3 tools/bulletml_converter.py patterns_xml/ patterns_blb/

# Convert back for validation
python3 tools/bulletml_converter.py pattern.blb pattern_verify.xml
```

## Requirements

- C++11 or later
- Existing BulletML library (bulletmlparser.h, bulletmlnode.h, etc.)
- Standard library: `<vector>`, `<string>`, `<cstdint>`, `<cstring>`, `<fstream>`

## Error Handling

```cpp
BulletMLParserBinary parser("pattern.blb");

try {
    parser.build();  // May throw BulletMLError
} catch (const BulletMLError& e) {
    std::cerr << "Parse error: " << e.what() << std::endl;
    // Also check: parser.getErrorMessage()
}
```

## Performance

Tested with 73 pattern files (boss + middle + zako):
- **XML total**: 79,914 bytes
- **BLB total**: 53,689 bytes  
- **Savings**: 26,225 bytes (32.8%)
- **Parse speed**: ~3-5x faster than XML parsing (no XML DOM overhead)
